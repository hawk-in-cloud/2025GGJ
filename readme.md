# 说明文档


1. 创建github仓库

* 首先克隆仓库（检查链接）
```
git clone https://github.com/hawk-in-cloud/2025GGJ.git
```

* 其次如何提交：
```
git init
git add readme.md
git add Assets/
git add Logs/
git add Packages/
git add ProjectSettings/
git add UserSettings/
git commit -m "提交说明"
git branch -M main
git remote add origin https://github.com/hawk-in-cloud/2025GGJ.git
git push -u origin main
```
